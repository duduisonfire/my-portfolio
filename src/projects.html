Agenda

Agenda is a contact list application where you can create an account and register your contacts, update and delete them. The application back-end was built in NodeJS.

This is the most complete and complex application in this portfolio, it is a CRUD application (CREATE, READ, UPDATE and DELETE) in a database. The front-end of the application was written with HTML, the Bootstrap CSS Framework and JavaScript. The Backend was written using NodeJS (JavaScript), ExpressJS, EJS (to render the pages) and Mongoose (to connect to the database). The database used was MongoDB (Non-relational and document-based).
